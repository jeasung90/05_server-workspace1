POST 방식으로 요청해보기 앙~

특징1. POST 방식으로 요청시 URL Body 영역에 데이터를 포함시켜서 넘김
=> 사용자가 입력한 값(데이터)들이 URL에 노출되지 않음
=> 즉, 로그인이나 회원가입 같은 경우 POST 방식에 적합함

특징2. Body 영역은 전송하는 길이에 제한이 없음
즉, 게시판 작성 같은 경우 POST 방식이 적합함
특징3. 즐겨찾기 가능하나(단순 url 저장) 데이터가 url에 노출되어있진 않음 => 단, 기존에 봤던 응답화면 볼 수 없을 것.
=> 즉, 검색기능 같은 경우에는 post 방식 부적 특징4. Time out이 존재하기 때문에 페이지 요청, 기다리는 시간이 있다.
